# NinoBaby E-Commerce

A modern, warm, and corporate e-commerce landing page for ninobaby.com featuring premium baby products.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
